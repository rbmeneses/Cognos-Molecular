# Cognos-Molecular
Cognos Molecular AI é uma aplicação avançada desenvolvida em Python e Streamlit que combina a precisão da quimioinformática (RDKit) com o poder da Inteligência Artificial Generativa (Google Gemini). Esta ferramenta permite analisar, gerar, visualizar e catalogar estruturas moleculares, desde compostos conhecidos até moléculas teóricas/hipotéticas
